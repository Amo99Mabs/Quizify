# 🇿🇦 Quizify

Quizify is a fun and interactive command-line quiz game focused on South African history, geography, and culture. Built with Python, it challenges users with multiple-choice questions and gives instant feedback along with a final score.

## 📚 Features

- 🧠 Five culturally themed multiple-choice questions
- ⌨️ Accepts user input and tracks guesses
- ✅ Validates answers and displays feedback
- 📊 Calculates and shows final percentage score
- 📋 Summary of correct answers and user responses
